# Lead Generation Scraper

Python tool for generating targeted business leads using Google Maps API and web scraping.

## Features
- Search businesses by keyword and location
- Extract:
  - Company name
  - Website
  - Phone numbers
  - Emails (from website)
  - Instagram links
- Remove duplicates
- Export to Excel

## Technologies
- Python
- Google Maps API
- BeautifulSoup
- Pandas

## Usage
1. Enter city (e.g. Oslo)
2. Enter business type (e.g. Beauty salon)
3. Get ready-to-use leads in Excel

## Output
Excel file with:
- Company
- Website
- Phone
- Emails
- Instagram

---

⚠️ Note: Requires your own Google Maps API key
